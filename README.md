# Hortus

Gardening planner, monitoring and automation solution.

## Features

- Manage a collection crops.
- Create culture of crops.
- Manage the differents phases of the culture; seeding, planting, harvesting.
- Create your gardenning spaces, plants and pots in a visual environment.
- Automate and monitor your garden and greenhouse.

## Components

- Server [Hortus Server](https://github.com/alexandrelamberty/hortus-server)
- Web interface [Hortus Web Application](https://github.com/alexandrelamberty/hortus-webapp)
- Sensors [Hortus-Wireless-Humidity-Temperature-Sensor](https://github.com/alexandrelamberty/hortus-wireless-temperature-sensor/)
- Controllers [Hortus-Controller](https://github.com/alexandrelamberty/hortus-controller)
